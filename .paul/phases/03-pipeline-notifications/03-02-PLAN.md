<objective>
## Goal
Surface pipeline data on the dashboard and provide a dedicated Alerts page showing all projects with upcoming or overdue next actions. Add an Alerts nav item to the sidebar with a live overdue badge count.

## Purpose
Completes Phase 3. The AYS team can now see their full pipeline status at a glance on the dashboard, and navigate to a single Alerts page to see every project where action is needed — ordered by urgency, overdue items highlighted in red.

## Output
- `/dashboard/alerts` page — sorted list of all projects with next_action_due_at set; overdue highlighted red
- Dashboard: "Needs Attention" stat card (overdue count, clickable to /dashboard/alerts) + pipeline stage breakdown section
- Sidebar "Alerts" nav item with red badge showing overdue count
- `AlertProject` type for the joined projects+clients+stages query
- `getAlerts()` and `getAlertCount()` server actions
</objective>
